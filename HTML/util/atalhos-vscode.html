Atalhos no vscode aprendidos com o curso

  1. Shift + Alt + F para formatar o código

  2. Shift + Alt + ↓ para copirar a linha para baixo

  3. ctrl + D para selecionar a próxima ocorrência da palavra selecionada
  4. ctrl + s para salvar o arquivo
  5. alt + z para quebrar a linha automaticamente, ou seja, quando o texto ultrapassar a largura da tela, ele irá quebrar a linha automaticamente
