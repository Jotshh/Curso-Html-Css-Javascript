Se repararmos bem, a especificidade do estilo inline é maior que a do CSS incorporado,

ou seja, o estilo inline tem prioridade sobre o CSS incorporado, e por isso, o título "Titulo estilizado inline" será exibido em vermelho, enquanto o título "Titulo estilizado com CSS incorporado" será exibido em azul.
