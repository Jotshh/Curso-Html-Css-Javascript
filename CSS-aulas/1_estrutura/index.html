Aula 03 - Notação decimal e notação hexadecimal de cor

A notação decimal e a notação hexadecimal são duas formas diferentes de representar cores em CSS. A notação decimal utiliza valores numéricos para representar as componentes de cor (vermelho, verde e azul), enquanto a notação hexadecimal utiliza uma combinação de números e letras para representar as mesmas componentes de cor. Ambas as notações são amplamente utilizadas em CSS para definir cores em elementos HTML, permitindo que os desenvolvedores escolham a forma que melhor se adequa às suas necessidades e preferências.

Além disso, temos o rgba, que é uma notação que permite definir a cor utilizando os valores de vermelho, verde, azul e ALFA (opacidade). O valor alfa varia de 0 a 1, onde 0 representa totalmente transparente e 1 representa totalmente opaco. Essa notação é útil para criar efeitos de transparência em elementos HTML, permitindo que o fundo ou outros elementos sejam visíveis através do elemento com a cor definida em rgba.
